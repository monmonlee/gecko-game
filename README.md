# 守宮陪伴 🦎

一款療癒系肥尾守宮飼養放置遊戲。牠是夜行性的——白天大部分時間在睡覺，關燈後才出來活動。你可以用夜視鏡靜靜看牠過自己的生活；隨著好感度提升，牠會越來越信任你，願意在你面前自在地睡覺，甚至爬上你的手心。

**▶ 開始玩：https://monmonlee.github.io/gecko-game/**

以真實飼養的肥尾守宮（African Fat-tailed Gecko）為原型。
